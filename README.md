# 中国 K-12 AI 科技素质教育平台 · 战略评估报告

一份以**麦肯锡标准**撰写的商业战略评估：面向**中国 K-12**、以 **AI 编程为旗舰**、扩展到机器人/科学/AI 素养等**非学科类科技素养**的学习平台，用免费内容获客、**家长会员**变现（AI 算力封装在会员额度内、对用户不可见）。以单页 GitHub Pages 网站形式输出，方便分享给合伙人与投资人。

> 核心结论：模式成立；但客户是中国 K-12 孩子——决定成败的不是 Token 毛利，而是「**家长信任 × 学习成效 × 双减合规**」。**第 13 章基于一手监管文件与券商研究做了实证修正与融资优化建议。**

## 🚀 发布到 GitHub Pages（约 1 分钟）

1. 把本仓库推送到 GitHub：
   ```bash
   git add .
   git commit -m "Add Vibe Coding strategy assessment report"
   git push
   ```
2. 打开仓库 **Settings → Pages**。
3. 在 **Build and deployment → Source** 选择 **Deploy from a branch**。
4. **Branch** 选择 `main`，文件夹选择 **`/docs`**，点 **Save**。
5. 等待约 1 分钟，访问：`https://<你的用户名>.github.io/<仓库名>/`

## 👀 本地预览

直接用浏览器打开 `docs/index.html` 即可；或起一个本地服务器：

```bash
# Python
python -m http.server 8000 --directory docs
# 然后访问 http://localhost:8000
```
